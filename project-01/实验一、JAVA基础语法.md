# Java 基础语法综合实验

## 一、实验目的

本实验旨在帮助大一学生掌握 Java 语言基础语法的核心应用，通过实操训练强化代码编写与调试能力。通过本实验，学生将：

* 熟练掌握 Java 类的定义规范、文件名与公共类的命名规则，理解 Java 程序的基本结构。
* 灵活运用基本数据类型、运算符（算术、关系、逻辑、位运算等），掌握类型转换的核心逻辑。
* 熟练使用分支结构（if-else、switch）与循环结构（for、while、do-while），掌握流程控制的设计方法。
* 理解Java运算符、break/continue 关键字的作用，能够解决综合类的逻辑控制问题。
* 规范编写代码注释，养成良好的 Java 编程习惯，为后续面向对象编程奠定基础。

## 实验原理

**程序结构原理**：Java 程序必须以类为载体，公共类名需与文件名完全一致，程序执行入口为固定的public static void main(String[] args)方法。

**数据类型与运算原理**：Java 数据类型分为基本数据类型（8 种）与引用数据类型，基本数据间存在自动类型转换（小转大）与强制类型转换（大转小）规则；运算符则根据功能分为算术、关系、逻辑、位运算等，不同运算符遵循优先级与结合性规则。

**流程控制原理**：分支结构通过条件判断执行不同代码逻辑，switch 结构基于常量值匹配分支；循环结构通过条件控制代码重复执行，break 用于终止循环 / 分支，continue 用于跳过本次循环。

### 具体实验题目

#### 题目 1：Hello World 基础改造（基础题）

1. 创建类HelloMyName，要求文件名与 public 类名完全一致。
2. 在main方法中输出两行内容：第一行Hello World!，第二行我的名字是：XXX（替换为自身姓名）。
3. 为main方法添加文档注释，说明方法作用、参数含义。

#### 题目 2：变量声明与算术运算（基础题）

1. 创建类VariableCalculator，在main方法中声明变量：整型a=18、b=5，浮点型c=3.8f、d=2.5。
2. 计算并输出a+b、a-b、a*b、a/b（整数除法）、a%b的结果。
3. 计算并输出c+d、(a+b)*c的结果，保留浮点型精度。
4. 声明变量int i=10; int j; ，并通过i++和++i分别对j进行赋值，输出运算后j的值与计算结果。

#### 题目 3：布尔运算与条件判断（基础题）

1. 创建类BooleanCondition，声明布尔变量flag1=true、flag2=false，依次计算并输出!flag1、flag1&flag2、flag1|flag2、flag1^flag2、flag1&&flag2、flag1||flag2的结果。
2. 声明整型变量score=78，通过 if-else 多分支结构判断成绩等级：≥90 为 “优秀”，70≤score<90 为 “良好”，60≤score<70 为 “及格”，<60 为 “不及格”。
3. 修改score值（55、92、66），测试不同分支的执行结果。
