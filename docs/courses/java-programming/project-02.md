# Project 02：类与对象

本项目覆盖类定义、对象创建、成员变量、成员方法、引用传递和简单类协作。

## 实验目的

通过类与对象综合实验，理解类的抽象方式、对象实例化、封装、构造方法重载、`static` 成员、参数传递和内部类。实验目标是把基础语法推进到面向对象建模和调试。

## 核心知识点

- 类定义属性和行为，对象是类通过 `new` 创建的实例。
- 封装通过 `private` 字段和 `getter/setter` 控制访问，并可在 setter 中校验数据。
- 构造方法与类同名，支持重载；显式定义构造方法后默认无参构造不再自动生成。
- `static` 成员属于类本身，被所有实例共享；静态方法不能直接访问实例成员。
- Java 参数传递本质是值传递，基本类型传值副本，引用类型传对象地址副本。
- 静态嵌套类和非静态内部类对外部类实例的依赖不同。

## 任务概览

1. 补全学生成绩管理系统，完成属性、构造方法、setter 校验、平均分计算和信息输出。
2. 练习类成员与实例成员，区分类变量、实例变量、静态方法和实例方法。
3. 通过点坐标、图书馆、学生等类观察对象引用赋值和参数传递。
4. 完成填空代码与思考题，解释引用复制、对象复制和 `==` 的差异。
