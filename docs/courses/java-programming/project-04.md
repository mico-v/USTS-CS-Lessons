# Project 04：数组与工具类

本项目整理数组初始化、数组访问、`Arrays` 工具类和数组引用行为相关练习。

## 实验目的

通过数组相关练习掌握 Java 数组的声明、初始化、遍历、工具类调用和引用传递行为。项目延续前几次实验的“观察输出、解释现象”方式，帮助理解数组作为引用类型时的内存语义。

## 核心知识点

- 数组长度固定，创建后通过下标访问元素。
- 数组遍历可使用普通 `for` 循环或增强 `for` 循环。
- `Arrays` 工具类可用于排序、查找、填充和字符串化输出。
- 数组变量保存引用，赋值和参数传递时需要区分“复制引用”和“复制内容”。

## 任务概览

1. 完成数组初始化和下标访问练习。
2. 使用 `Arrays` 工具类处理数组。
3. 观察数组引用赋值、方法传参和元素修改带来的结果变化。
