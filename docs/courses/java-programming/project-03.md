# Project 03：继承、重写与重载

本项目围绕构造方法调用链、`this`、方法重写和方法重载展开。

## 实验目的

通过 Java 综合实验理解继承关系中的构造方法执行顺序、`this` 与 `super` 的语义、方法重写和方法重载的区别。实验强调用小型继承体系观察运行期动态绑定和编译期静态绑定。

## 核心知识点

- Java 通过 `extends` 实现继承，子类可以复用父类非私有成员。
- 创建子类对象时先执行父类构造方法，再执行子类构造方法。
- `super(...)` 必须写在构造方法第一行，用于调用父类构造方法。
- 方法重写发生在父子类之间，方法签名相同，运行期动态绑定。
- 方法重载发生在同一类或继承体系中，方法名相同但参数列表不同，编译期静态绑定。
- `final` 方法可以继承但不能被重写。

## 任务概览

1. 在 `Vehicle -> Car -> ElectricCar` 三层继承体系中完成构造方法链。
2. 在 `Shape -> Circle` 继承体系中重写信息展示、面积计算和描述方法。
3. 在 `Calculator -> ScientificCalculator` 中对比重写与重载，并通过 `super` 与 `this` 观察调用差异。
