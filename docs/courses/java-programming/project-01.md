# Project 01：Java 基础语法

本项目覆盖 Java 程序结构、基本数据类型、运算符、分支、循环、流程控制、类型转换和 `instanceof`。

## 实验目的

通过基础语法综合实验，训练 Java 类定义、文件名与公共类命名规则、程序入口、数据类型转换、运算符优先级、分支结构和循环结构。实验强调规范注释和基础调试能力，为后续面向对象编程打基础。

## 核心知识点

- Java 程序必须以类为载体，`public` 类名需要与文件名一致。
- `main` 方法是程序入口，基本数据类型之间存在自动类型转换和强制类型转换。
- 运算符包括算术、关系、逻辑、位运算等类别，需要注意优先级和结合性。
- `if-else`、`switch`、`for`、`while`、`do-while` 用于组织流程控制。
- `break` 用于终止循环或分支，`continue` 用于跳过本次循环。

## 任务概览

1. 输出 `Hello World` 和个人姓名，补充 `main` 方法文档注释。
2. 声明整型、浮点型变量，完成算术运算、取模和自增运算观察。
3. 练习布尔运算，并用多分支结构判断成绩等级。
4. 使用 `switch` 判断季节，用不同循环输出奇偶数和累加结果。
5. 使用 `continue` 与 `break` 控制循环，并计算指定范围内的整除求和。
6. 完成强制类型转换、自动类型转换和三元运算符求最大值。
7. 选做位运算和 `instanceof` 判断，观察对象类型关系。
