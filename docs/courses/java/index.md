# Java 与面向对象程序设计

本课程页合并原来的 Java 程序设计与面向对象程序设计内容，按课程性质分为两部分：

- [Java 实践课](./practice/)：原 Java 程序设计项目与实验内容，侧重编程任务和项目实践。
- [OOP 理论课](./theory/)：原面向对象程序设计复习讲义，侧重 Java 语法、面向对象概念和期末复习。

## 课程内容

| 分类 | 内容 | 入口 |
| --- | --- | --- |
| Java 实践课 | Project 01-04，覆盖 Java 基础、类与对象、继承、数组与工具类 | [进入实践课](./practice/) |
| OOP 理论课 | 10 份专题复习讲义，覆盖 Java 基础、封装、继承、多态、集合、异常、I/O、线程 | [进入理论课](./theory/) |

## 使用建议

先通过理论课建立语法和面向对象知识框架，再用实践课项目检查自己是否能把概念写成可运行代码。
