# Java 实践课

## 项目目录

| 项目 | 主题 | 内容摘要 |
| --- | --- | --- |
| [Project 01](./project-01.md) | Java 基础语法 | 程序结构、数据类型、运算符、分支循环 |
| [Project 02](./project-02.md) | 类与对象 | 封装、构造方法、static、参数传递、内部类 |
| [Project 03](./project-03.md) | 继承、重写与重载 | 继承链、this/super、Override 与 Overload |
| [Project 04](./project-04.md) | 数组与工具类 | 数组初始化、访问、工具类和引用行为 |

## 使用方式

每个项目页以实验说明为主，适合快速回顾课程内容和任务范围。
