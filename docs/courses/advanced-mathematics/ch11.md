# 第十一章 曲线积分与曲面积分 — 期末复习讲义

> 写给初学者的话：这一章把积分从"直线/平面/立体"推广到"曲线"和"曲面"上。每种积分都有两类（第一型看"大小"，第二型看"方向"），然后用格林公式和高斯公式把它们联系起来。核心线索：**定义→计算→公式→应用**。

---

## 一、对弧长的曲线积分（第一型曲线积分）

### 1.1 直觉理解

想象一根弯曲的金属丝，每处的密度不一样 $\rho(x,y)$。求整根丝的质量？把丝切成小段 → 每段近似为均匀 → 求和 → 取极限。

### 1.2 定义

$$\int_L f(x,y)\,ds = \lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i, \eta_i) \cdot \Delta s_i$$

其中 $\Delta s_i$ 是第 $i$ 段弧的长度。**关键：$ds$ 是弧长元素，永远为正。**

### 1.3 计算公式（必背）

曲线 $L$ 的参数方程 $x = \varphi(t),\ y = \psi(t),\ \alpha \leq t \leq \beta$（要求 $\alpha < \beta$）：

$$\boxed{\int_L f(x,y)\,ds = \int_\alpha^\beta f(\varphi(t), \psi(t)) \cdot \sqrt{\varphi'(t)^2 + \psi'(t)^2}\,dt}$$

**记忆要点**：$ds = \sqrt{dx^2 + dy^2} = \sqrt{\varphi'^2 + \psi'^2}\,dt$

**特殊情况**：
- 若 $y = \psi(x),\ a \leq x \leq b$：$ds = \sqrt{1 + \psi'(x)^2}\,dx$
- 若 $x = \varphi(y),\ c \leq y \leq d$：$ds = \sqrt{1 + \varphi'(y)^2}\,dy$
- 空间曲线 $\Gamma: x=\varphi(t), y=\psi(t), z=\omega(t)$：$ds = \sqrt{\varphi'^2 + \psi'^2 + \omega'^2}\,dt$

> **注意**：第一型曲线积分与方向无关（$\int_L = \int_{-L}$），因为弧长总是正的。

### 1.4 典型题

**例1**：$\int_L xy\,ds$，$L$ 为椭圆 $x=a\cos t,\ y=b\sin t$ 在第一象限部分。

$$ds = \sqrt{a^2\sin^2 t + b^2\cos^2 t}\,dt$$

$$I = \int_0^{\pi/2} ab\cos t \sin t \sqrt{a^2\sin^2 t + b^2\cos^2 t}\,dt = \frac{ab(a^2+ab+b^2)}{3(a+b)}$$

**例2**：$\int_L y\,ds$，$L$ 为抛物线 $y=x^2$ 从 $(0,0)$ 到 $(1,1)$。

$$I = \int_0^1 x^2 \sqrt{1+4x^2}\,dx = \frac{5\sqrt{5}-1}{12}$$

**例3**（闭合曲线）：$\oint_L (x+y)\,ds$，$L$ 为三角形 $(0,0),(1,0),(1,1)$。

分成三段 $OA + AB + BO$，逐段计算：$I = 2 + \sqrt{2}$

## 二、对坐标的曲线积分（第二型曲线积分）

### 2.1 直觉理解

变力 $\vec{F} = P(x,y)\vec{i} + Q(x,y)\vec{j}$ 沿曲线 $L$ 从 $A$ 移动到 $B$，求做功？

$$W = \int_L P\,dx + Q\,dy$$

### 2.2 定义

$$\int_L P(x,y)\,dx + Q(x,y)\,dy = \lim_{\lambda \to 0} \sum_{i=1}^{n} [P(\xi_i,\eta_i)\Delta x_i + Q(\xi_i,\eta_i)\Delta y_i]$$

> **关键区别**：第一型看弧长（标量），第二型看坐标投影（向量），**方向有关**！

### 2.3 方向性（必背）

$$\int_{-L} P\,dx + Q\,dy = -\int_L P\,dx + Q\,dy$$

**反向变号**——这是与第一型曲线积分最重要的区别。

### 2.4 计算公式（必背）

参数方程 $x=\varphi(t),\ y=\psi(t)$，$t$ 从 $\alpha$（起点）到 $\beta$（终点）：

$$\boxed{\int_L P\,dx + Q\,dy = \int_\alpha^\beta [P(\varphi,\psi)\varphi'(t) + Q(\varphi,\psi)\psi'(t)]\,dt}$$

**特殊情况**：
- $y = y(x),\ x$ 从 $a$ 到 $b$：$\int_a^b [P(x,y(x)) + Q(x,y(x))y'(x)]\,dx$

### 2.5 两类曲线积分的联系

$$\int_L P\,dx + Q\,dy = \int_L [P\cos\alpha + Q\cos\beta]\,ds$$

其中 $\cos\alpha = \frac{dx}{ds},\ \cos\beta = \frac{dy}{ds}$ 是切向量的方向余弦。

### 2.6 典型题

**例**：$\int_L xy\,dx$，$L$ 为抛物线 $y^2=x$ 从 $A(1,-1)$ 到 $B(1,1)$。

以 $y$ 为参数，$x=y^2$，$dx=2y\,dy$：

$$I = \int_{-1}^{1} y^2 \cdot y \cdot 2y\,dy = 2\int_{-1}^{1} y^4\,dy = \frac{4}{5}$$

**路径无关的发现**：$\int_L 2xy\,dx + x^2\,dy$ 沿三条不同路径从 $(0,0)$ 到 $(1,1)$，结果都是 $1$！这引出下一节的"路径无关"。

## 三、格林公式（核心重点）

### 3.1 基本概念

- **单连通区域**：没有"洞"的区域（任何闭曲线围住的都在区域内）
- **复连通区域**：有"洞"的区域
- **正向边界**：沿边界走时区域在左手边（外边界逆时针，内边界顺时针）

### 3.2 格林公式（必背）

$$\boxed{\oint_L P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dx\,dy}$$

其中 $L$ 是 $D$ 的正向边界，$P, Q$ 在 $D$ 上有连续一阶偏导数。

> **通俗理解**：格林公式把曲线积分变成了二重积分！

### 3.3 格林公式的三大应用

#### 应用1：简化闭曲线积分

直接套公式，把曲线积分化为二重积分。

**例**：$\oint_L (y^2+xe^{2y})dx + (x^2e^{2y}-x^2)dy$，$L: (x-2)^2+y^2=4$（正向）

$$\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 2xe^{2y} - 2y - 2xe^{2y} = -2y$$

不对，重新算：$Q = x^2 e^{2y} - x^2$，$\frac{\partial Q}{\partial x} = 2xe^{2y} - 2x$

$P = y^2 + xe^{2y}$，$\frac{\partial P}{\partial y} = 2y + 2xe^{2y}$

差 $= -2x - 2y$，在圆域上积分得 $-16\pi$

#### 应用2：补线法（非闭曲线积分）

曲线不闭合时，**添加辅助线段**使其闭合，用格林公式，再减去辅助线上的积分。

**例**：$\int_L (x^2+3y)dx + (y^2-x)dy$，$L$ 为上半圆 $y=\sqrt{4x-x^2}$ 从 $O(0,0)$ 到 $A(4,0)$。

补线段 $AO$（从 $A$ 到 $O$ 沿 $x$ 轴），使 $L + AO$ 构成正向闭曲线。

$$\oint = \iint_D (-1-3)\,dxdy = -4 \cdot \text{半圆面积} = -4 \cdot 2\pi = -8\pi$$

$$\int_{AO} = \int_4^0 x^2\,dx = -\frac{64}{3}$$

$$\int_L = -8\pi - \left(-\frac{64}{3}\right) = -8\pi + \frac{64}{3}$$

#### 应用3：计算平面面积

取 $P=-y,\ Q=x$，则 $\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 2$：

$$\boxed{A = \frac{1}{2}\oint_L x\,dy - y\,dx}$$

**例**：椭圆 $x=a\cos\theta,\ y=b\sin\theta$ 的面积：

$$A = \frac{1}{2}\int_0^{2\pi} (a\cos\theta \cdot b\cos\theta + b\sin\theta \cdot a\sin\theta)\,d\theta = \pi ab$$

## 四、曲线积分与路径无关（核心重点）

### 4.1 四个等价条件（必背）

在**单连通**区域 $D$ 上，$P, Q$ 有连续一阶偏导数，以下四条**等价**：

| 编号 | 条件 | 含义 |
|------|------|------|
| (1) | $\oint_L P\,dx + Q\,dy = 0$（任意闭曲线） | 闭路上积分为零 |
| (2) | $\int_L P\,dx + Q\,dy$ 与路径无关 | 只与起点终点有关 |
| (3) | $P\,dx + Q\,dy = du$（全微分） | 存在原函数 $u(x,y)$ |
| (4) | $\dfrac{\partial P}{\partial y} = \dfrac{\partial Q}{\partial x}$ | **最常用的判定条件** |

> **考试核心**：看到 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$，立刻想到"路径无关"！

### 4.2 求原函数

$$u(x,y) = \int_{x_0}^{x} P(x, y_0)\,dx + \int_{y_0}^{y} Q(x, y)\,dy$$

或

$$u(x,y) = \int_{y_0}^{y} Q(x_0, y)\,dy + \int_{x_0}^{x} P(x, y)\,dx$$

### 4.3 典型题

**例1**：$\int_L (1+xe^{2y})dx + (x^2e^{2y}-y)dy$，$L$ 为上半圆从 $O(0,0)$ 到 $A(4,0)$。

验证：$\frac{\partial P}{\partial y} = 2xe^{2y} = \frac{\partial Q}{\partial x}$ ✓ 路径无关

换为直线 $OA$（$y=0,\ dy=0$）：$I = \int_0^4 (1+x)\,dx = 12$

**例2**：$I = \int_L \frac{(x-y)dx + (x+y)dy}{x^2+y^2}$，$L: y=2-2x^2$ 从 $A(-1,0)$ 到 $B(1,0)$。

验证 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$（原点除外）。换路径为上半单位圆弧：$I = -\pi$

**例3**：验证 $xy^2\,dx + x^2y\,dy$ 是全微分并求原函数。

$\frac{\partial P}{\partial y} = 2xy = \frac{\partial Q}{\partial x}$ ✓

$$u = \int_0^x 0\,dx + \int_0^y x^2 y\,dy = \frac{1}{2}x^2y^2$$

## 五、对面积的曲面积分（第一型曲面积分）

### 5.1 定义

$$\iint_\Sigma f(x,y,z)\,dS = \lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i, \eta_i, \zeta_i) \cdot \Delta S_i$$

$dS$ 是面积元素，**与方向无关**。

### 5.2 计算公式（必背）

曲面 $z = z(x,y)$，$(x,y) \in D_{xy}$：

$$\boxed{\iint_\Sigma f(x,y,z)\,dS = \iint_{D_{xy}} f(x, y, z(x,y)) \cdot \sqrt{1 + z_x^2 + z_y^2}\,dx\,dy}$$

> **记忆**：和二重积分几乎一样，只多了一个 $\sqrt{1+z_x^2+z_y^2}$ 因子。

### 5.3 典型题

**例**：$\iint_\Sigma \frac{dS}{z}$，$\Sigma$ 为球面 $x^2+y^2+z^2=a^2$ 被平面 $z=h$（$0<h<a$）截出的顶部。

$$z = \sqrt{a^2-x^2-y^2},\quad \sqrt{1+z_x^2+z_y^2} = \frac{a}{\sqrt{a^2-x^2-y^2}}$$

$$I = \iint_{D} \frac{a}{a^2-x^2-y^2}\,dx\,dy = 2\pi a \ln\frac{a}{h}$$

## 六、对坐标的曲面积分（第二型曲面积分）

### 6.1 有向曲面

给曲面指定一个"正面"（法向量方向）。法向量 $\vec{n} = (\cos\alpha, \cos\beta, \cos\gamma)$。

- $\cos\gamma > 0$：取上侧
- $\cos\gamma < 0$：取下侧

### 6.2 定义

$$\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy$$

**向量形式**：$\iint_\Sigma \vec{A} \cdot d\vec{S}$，其中 $d\vec{S} = \vec{n}\,dS = (dydz, dzdx, dxdy)$

> **关键区别**：第二型与方向有关，**反向变号**。

### 6.3 计算公式（必背）

曲面 $z = z(x,y)$，取上侧：

$$\boxed{\iint_\Sigma R(x,y,z)\,dxdy = +\iint_{D_{xy}} R(x, y, z(x,y))\,dxdy}$$

取下侧则加负号。类似地：
- $x = x(y,z)$，取前侧（$\cos\alpha > 0$）：$\iint P\,dydz = +\iint_{D_{yz}} P(x(y,z), y, z)\,dydz$
- $y = y(z,x)$，取右侧（$\cos\beta > 0$）：$\iint Q\,dzdx = +\iint_{D_{zx}} Q(x, y(z,x), z)\,dzdx$

### 6.4 两类曲面积分的联系

$$\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iint_\Sigma (P\cos\alpha + Q\cos\beta + R\cos\gamma)\,dS$$

---

## 七、高斯公式（核心重点）

### 7.1 高斯公式（必背）

$$\boxed{\oiint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iiint_\Omega \left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right) dx\,dy\,dz}$$

其中 $\Sigma$ 是 $\Omega$ 的**外表面**（取外侧），$P, Q, R$ 在 $\Omega$ 上有连续一阶偏导数。

> **通俗理解**：高斯公式把封闭曲面上的曲面积分变成了三重积分！（类似格林公式在三维的推广）

### 7.2 典型题

**例1**：$\oiint_\Sigma (y-z)x\,dydz + 0\,dzdx + (x-y)\,dxdy$，$\Sigma$ 为柱面 $x^2+y^2=1$ 与 $z=0, z=3$ 围成的闭曲面（外侧）。

$$\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} = (y-z) + 0 + 0 = y-z$$

$$\iiint_\Omega (y-z)\,dv$$

由对称性 $\iiint y\,dv = 0$（$y$ 是奇函数），$\iiint z\,dv = \bar{z} \cdot V$

**例2**（补面法）：$\iint_\Sigma x^2\cos\alpha + y^2\cos\beta + z^2\cos\gamma\,dS$，$\Sigma$ 为锥面 $x^2+y^2=z^2$（$0 \leq z \leq h$），取下侧。

锥面不是闭曲面，补上顶面 $\Sigma_1: z=h$（取上侧）。用高斯公式：

$$\iiint_\Omega (2x+2y+2z)\,dv$$

减去 $\Sigma_1$ 上的积分，最终得 $-\frac{\pi h^4}{2}$。

### 7.3 散度（必背）

$$\text{div}\,\vec{A} = \nabla \cdot \vec{A} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$$

高斯公式可写成：$\oiint_\Sigma \vec{A} \cdot d\vec{S} = \iiint_\Omega \text{div}\,\vec{A}\,dv$

> **物理意义**：$\text{div}\,\vec{A} > 0$ 表示该点有"源"（流体涌出），$< 0$ 表示"汇"（流体被吸收），$= 0$ 表示无源场。

### 7.4 闭曲面积分为零的条件

在**空间二维单连通**区域 $G$ 中：

$$\oiint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = 0 \quad \Longleftrightarrow \quad \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} = 0$$

## 八、考试公式速查表

| 公式 | 内容 | 用途 |
|------|------|------|
| 第一型曲线积分 | $\int_L f\,ds = \int_\alpha^\beta f \sqrt{\varphi'^2+\psi'^2}\,dt$ | 质量、弧长 |
| 第二型曲线积分 | $\int_L P\,dx+Q\,dy = \int_\alpha^\beta [P\varphi'+Q\psi']\,dt$ | 做功 |
| **格林公式** | $\oint_L P\,dx+Q\,dy = \iint_D (\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y})\,dxdy$ | 曲线→二重 |
| 路径无关判定 | $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$ | 换路径简化 |
| 面积公式 | $A = \frac{1}{2}\oint_L x\,dy - y\,dx$ | 平面面积 |
| 第一型曲面积分 | $\iint_\Sigma f\,dS = \iint_D f\sqrt{1+z_x^2+z_y^2}\,dxdy$ | 曲面质量 |
| 第二型曲面积分 | $\iint_\Sigma R\,dxdy = \pm\iint_D R\,dxdy$ | 通量 |
| **高斯公式** | $\oiint_\Sigma P\,dydz+Q\,dzdx+R\,dxdy = \iiint_\Omega \text{div}\,\vec{A}\,dv$ | 曲面→三重 |
| 散度 | $\text{div}\,\vec{A} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$ | 源/汇判断 |

## 九、易错点提醒

1. **第一型 vs 第二型方向问题**：第一型与方向无关（弧长恒正），第二型反向变号。

2. **格林公式要求闭曲线**：不闭合时要**补线**，算完后再减去补线的部分。

3. **格林公式中 $P$ 和 $Q$ 在区域内不能有奇点**：如果 $\frac{\partial P}{\partial y}$ 或 $\frac{\partial Q}{\partial x}$ 在区域内某点不存在（如原点处分母为零），不能直接用格林公式，需要挖洞处理。

4. **高斯公式要求闭曲面**：不封闭时要**补面**（如加一个平面盖子）。

5. **第二型曲面积分的正负号**：上侧取正，下侧取负；前侧取正，后侧取负；右侧取正，左侧取负。

6. **两类积分联系的方向余弦**：$\cos\alpha, \cos\beta, \cos\gamma$ 是**单位**法向量的分量，要归一化。

## 十、本章知识框架

```
曲线积分与曲面积分
├── 曲线积分
│   ├── 第一型（对弧长）：∫f ds，与方向无关
│   ├── 第二型（对坐标）：∫Pdx+Qdy，反向变号
│   └── 两类联系：∫Pdx+Qdy = ∫(Pcosα+Qcosβ)ds

│   ├── 公式：∮Pdx+Qdy = ∬(∂Q/∂x - ∂P/∂y)dxdy
│   ├── 应用：简化计算、补线法、面积公式
│   └── 路径无关：∂P/∂y = ∂Q/∂x ↔ 四个等价条件
├── 曲面积分
│   ├── 第一型（对面积）：∬f dS，与方向无关
│   ├── 第二型（对坐标）：∬Pdydz+Qdzdx+Rdxdy，反向变号
│   └── 两类联系：∬A·dS = ∬(Pcosα+Qcosβ+Rcosγ)dS
└── 高斯公式
    ├── 公式：∯A·dS = ∭div A dv
    ├── 补面法
    ├── 散度：div A = ∂P/∂x + ∂Q/∂y + ∂R/∂z
    └── 闭曲面积分为零的条件
```
