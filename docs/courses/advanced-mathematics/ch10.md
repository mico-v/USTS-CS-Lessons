# 第十章 重积分 — 期末复习讲义

> 写给初学者的话：重积分就是把"定积分"从一维推广到二维（二重积分）和三维（三重积分）。核心思想只有一句话：**分割→近似→求和→取极限**。计算的关键是把重积分化成多次定积分（累次积分），一步一步算。

---

## 一、二重积分的概念

### 1.1 直觉理解

想象你有一块形状不规则的蛋糕（底面是区域 $D$），顶面是曲面 $z = f(x,y)$。你想知道这块蛋糕的体积。

做法：把底面切成很多小方块 → 每个小方块上的蛋糕近似看成小长方体 → 把所有小长方体体积加起来 → 切得越细越精确 → 取极限就是二重积分。

### 1.2 正式定义

$$\iint_D f(x,y)\,d\sigma = \lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i, \eta_i) \cdot \Delta\sigma_i$$

其中 $\lambda$ 是所有小区域中最大直径，$\Delta\sigma_i$ 是第 $i$ 个小区域的面积。

**关键术语**：被积函数 $f(x,y)$、积分区域 $D$、面积元素 $d\sigma$

### 1.3 几何意义（必背）

| 条件 | 几何意义 |
|------|----------|
| $f(x,y) \geq 0$ | 曲顶柱体在 $xOy$ 平面上方的体积 |
| $f(x,y) \leq 0$ | 体积的相反数（负值） |
| $f(x,y)$ 有正有负 | 上方面积 − 下方面积（代数和） |

## 二、二重积分的性质（必背7条）

| 编号 | 性质 | 公式 |
|------|------|------|
| 1 | 常数提取 | $\iint_D kf\,d\sigma = k\iint_D f\,d\sigma$ |
| 2 | 线性性 | $\iint_D (f \pm g)\,d\sigma = \iint_D f\,d\sigma \pm \iint_D g\,d\sigma$ |
| 3 | 区域可加性 | $D = D_1 \cup D_2 \Rightarrow \iint_D = \iint_{D_1} + \iint_{D_2}$ |
| 4 | 面积公式 | $\iint_D 1\,d\sigma = \sigma$（$\sigma$ 是 $D$ 的面积） |
| 5 | 比较性质 | $f \leq g \Rightarrow \iint_D f \leq \iint_D g$ |
| 6 | **估值不等式** | $m\sigma \leq \iint_D f\,d\sigma \leq M\sigma$（$m, M$ 是最值） |
| 7 | **中值定理** | $\exists (\xi,\eta) \in D$，使 $\iint_D f\,d\sigma = f(\xi,\eta) \cdot \sigma$ |

> **考试高频**：性质6（估值不等式）和性质7（中值定理）常用于估计积分大小或证明不等式。

### 典型题：估计积分

**例**：估计 $I = \iint_D e^{x^2+y^2}\,d\sigma$，其中 $D: \frac{x^2}{a^2} + \frac{y^2}{b^2} \leq 1$（$0 < b < a$）

**解**：在 $D$ 上，$0 \leq x^2 + y^2 \leq a^2$，所以 $1 \leq e^{x^2+y^2} \leq e^{a^2}$

$D$ 的面积 $\sigma = \pi ab$

由估值不等式：$\pi ab \leq I \leq e^{a^2} \cdot \pi ab$

## 三、二重积分的计算（核心考点）

### 3.1 直角坐标系下化为累次积分

**核心思想**：把二重积分变成两次定积分。

#### X型区域（先 $y$ 后 $x$）

区域特征：平行于 $y$ 轴的直线穿过 $D$，最多交边界2次。

$$D: a \leq x \leq b,\quad \varphi_1(x) \leq y \leq \varphi_2(x)$$

$$\iint_D f(x,y)\,d\sigma = \int_a^b \left[\int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy\right] dx$$

#### Y型区域（先 $x$ 后 $y$）

区域特征：平行于 $x$ 轴的直线穿过 $D$，最多交边界2次。

$$D: c \leq y \leq d,\quad \psi_1(y) \leq x \leq \psi_2(y)$$

$$\iint_D f(x,y)\,d\sigma = \int_c^d \left[\int_{\psi_1(y)}^{\psi_2(y)} f(x,y)\,dx\right] dy$$

> **初学者口诀**：先画区域图 → 确定积分顺序 → 写出上下限 → 从内到外逐层计算。

### 3.2 交换积分次序（必考题型）

**步骤**：
1. 根据原积分的上下限画出积分区域 $D$
2. 把 $D$ 用另一种方式描述（X型↔Y型互换）
3. 写出新次序的积分

### 典型题：交换积分次序

**例**：交换 $I = \int_0^1 dx \int_0^{1-x} f(x,y)\,dy$ 的积分次序

**解**：画图，区域 $D: 0 \leq x \leq 1,\ 0 \leq y \leq 1-x$

换成 Y 型：$0 \leq y \leq 1,\ 0 \leq x \leq 1-y$

$$I = \int_0^1 dy \int_0^{1-y} f(x,y)\,dx$$

### 典型题：被积函数含 $e^{-y^2}$ 必须换序

**例**：$\iint_D e^{-y^2}\,d\sigma$，$D$ 为 $(0,0),(1,1),(0,1)$ 围成的三角形

**解**：$e^{-y^2}$ 对 $y$ 没有初等原函数，必须先积 $x$。

Y型：$0 \leq y \leq 1,\ 0 \leq x \leq y$

$$I = \int_0^1 \left[\int_0^y e^{-y^2}\,dx\right] dy = \int_0^1 y e^{-y^2}\,dy = \frac{1}{2}\left(1 - \frac{1}{e}\right)$$

> **经验法则**：看到 $e^{-y^2}$、$e^{x/y}$、$\frac{\sin x}{x}$ 等无初等原函数的被积函数，一定要考虑交换次序。

## 四、三重积分

### 4.1 定义

$$\iiint_\Omega f(x,y,z)\,dv = \lim_{\lambda \to 0} \sum_{k=1}^{n} f(\xi_k, \eta_k, \zeta_k) \cdot \Delta v_k$$

其中 $dv = dx\,dy\,dz$（直角坐标体积元素）。

### 4.2 四种计算方法（核心考点）

#### 方法一：直角坐标化为三次积分（先一后二）

投影 $\Omega$ 到 $xOy$ 平面得 $D$，对 $D$ 中每一点 $(x,y)$，$z$ 的范围是 $z_1(x,y) \leq z \leq z_2(x,y)$：

$$\iiint_\Omega f\,dv = \iint_D \left[\int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z)\,dz\right] d\sigma$$

再把 $D$ 上的二重积分化为累次积分。

**例**：求 $\iiint_\Omega x\,dv$，$\Omega$ 为 $x+2y+z=1$ 与三个坐标面围成的四面体。

解：$z$ 从 $0$ 到 $1-x-2y$，$y$ 从 $0$ 到 $\frac{1-x}{2}$，$x$ 从 $0$ 到 $1$。

$$I = \int_0^1 dx \int_0^{\frac{1-x}{2}} dy \int_0^{1-x-2y} x\,dz = \frac{1}{48}$$

#### 方法二：截面法（先二后一）

投影到 $z$ 轴得区间 $[c_1, c_2]$，对每个 $z$，截面 $D_z$ 是二维区域：

$$\iiint_\Omega f\,dv = \int_{c_1}^{c_2} \left[\iint_{D_z} f(x,y,z)\,dx\,dy\right] dz$$

> **何时用截面法**：被积函数只含 $z$（或只含 $x$、$y$），截面面积容易算时特别方便。

**经典题**：求 $\iiint_\Omega z^2\,dv$，$\Omega$ 为椭球 $\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} \leq 1$

截面 $D_z$：$z$ 处的椭圆面积 $= \pi ab(1 - z^2/c^2)$

$$I = \int_{-c}^{c} z^2 \cdot \pi ab\left(1 - \frac{z^2}{c^2}\right) dz = \frac{4}{15}\pi abc^3$$

#### 方法三：柱面坐标（必背公式）

$$x = \rho\cos\theta,\quad y = \rho\sin\theta,\quad z = z$$

$$dv = \rho\,d\rho\,d\theta\,dz$$

$$\iiint_\Omega f\,dv = \iiint f(\rho\cos\theta, \rho\sin\theta, z)\,\rho\,d\rho\,d\theta\,dz$$

> **何时用柱面坐标**：积分区域或被积函数含 $x^2 + y^2$（即 $\rho^2$）时。

**典型题**：求 $\iiint_\Omega z\,dv$，$\Omega$ 由球面 $x^2+y^2+z^2=4$ 和抛物面 $x^2+y^2=3z$ 围成。

交线：$z=1,\ \rho=\sqrt{3}$

$$I = \int_0^{2\pi} d\theta \int_0^{\sqrt{3}} \rho\,d\rho \int_{\rho^2/3}^{\sqrt{4-\rho^2}} z\,dz = \frac{13\pi}{4}$$

#### 方法四：球面坐标（必背公式）

$$x = r\sin\varphi\cos\theta,\quad y = r\sin\varphi\sin\theta,\quad z = r\cos\varphi$$

$$dv = r^2\sin\varphi\,dr\,d\varphi\,d\theta$$

范围：$r \geq 0,\ 0 \leq \varphi \leq \pi,\ 0 \leq \theta \leq 2\pi$

> **何时用球面坐标**：积分区域含球面 $x^2+y^2+z^2=R^2$ 或锥面 $z=\sqrt{x^2+y^2}$（即 $\varphi = \pi/4$）时。

**典型题**：求 $\iiint_\Omega (x^2+y^2+z^2)\,dv$，$\Omega$ 由锥面 $z=\sqrt{x^2+y^2}$ 和球面 $x^2+y^2+z^2=R^2$ 围成。

锥面对应 $\varphi = \pi/4$，球面对应 $r = R$

$$I = \int_0^{2\pi} d\theta \int_0^{\pi/4} \sin\varphi\,d\varphi \int_0^R r^4\,dr = \frac{\pi R^5}{5}(2-\sqrt{2})$$

### 4.3 坐标选择速查表

| 区域特征 | 推荐坐标 |
|----------|----------|
| 长方体、四面体 | 直角坐标 |
| 含 $x^2+y^2$（圆柱、旋转抛物面） | **柱面坐标** |
| 含球面、锥面 | **球面坐标** |
| 截面面积好算，被积函数只含一个变量 | 截面法 |

## 五、重积分的应用

### 5.1 体积

$$V = \iint_D [z_2(x,y) - z_1(x,y)]\,d\sigma$$

**经典题**：两个等半径圆柱 $x^2+y^2=R^2$ 和 $x^2+z^2=R^2$ 围成的立体体积。

利用对称性取第一卦限：$V = 8\iint_D \sqrt{R^2-x^2}\,dx\,dy = \frac{16}{3}R^3$

### 5.2 曲面面积（必背公式）

曲面 $z = f(x,y)$ 在区域 $D$ 上的面积：

$$A = \iint_D \sqrt{1 + \left(\frac{\partial z}{\partial x}\right)^2 + \left(\frac{\partial z}{\partial y}\right)^2}\,dx\,dy$$

**例**：球面 $x^2+y^2+z^2=a^2$ 被柱面 $x^2+y^2=ax$ 所截部分的面积。

$$\sqrt{1+z_x^2+z_y^2} = \frac{a}{\sqrt{a^2-x^2-y^2}}$$

极坐标下计算得 $A = 4a^2(\pi - 2)$

### 5.3 质心（形心）

均匀薄片（密度为常数）的形心：

$$\bar{x} = \frac{1}{A}\iint_D x\,d\sigma, \qquad \bar{y} = \frac{1}{A}\iint_D y\,d\sigma$$

非均匀薄片（密度 $\rho(x,y)$）的质心：

$$\bar{x} = \frac{\iint_D x\rho\,d\sigma}{\iint_D \rho\,d\sigma}, \qquad \bar{y} = \frac{\iint_D y\rho\,d\sigma}{\iint_D \rho\,d\sigma}$$

### 5.4 转动惯量

$$I_x = \iint_D y^2 \rho(x,y)\,d\sigma, \qquad I_y = \iint_D x^2 \rho(x,y)\,d\sigma$$

**例**：直角三角形（直角边 $a, b$），关于沿 $y$ 轴的直角边的转动惯量：$I_y = \frac{1}{12}\rho a^3 b$

## 六、考试必背公式清单

### 体积元素对照

| 坐标系 | 体积元素 |
|--------|----------|
| 直角坐标 | $dv = dx\,dy\,dz$ |
| 柱面坐标 | $dv = \rho\,d\rho\,d\theta\,dz$ |
| 球面坐标 | $dv = r^2\sin\varphi\,dr\,d\varphi\,d\theta$ |

### 球面坐标变换

$$x^2+y^2+z^2 = r^2, \qquad x^2+y^2 = r^2\sin^2\varphi$$

锥面 $z = \sqrt{x^2+y^2}$ 对应 $\varphi = \dfrac{\pi}{4}$

$xOy$ 平面对应 $\varphi = \dfrac{\pi}{2}$

### 对称性简化（非常重要）

- 若 $\Omega$ 关于 $xOy$ 平面对称，$f(x,y,-z) = -f(x,y,z)$（奇函数），则 $\iiint_\Omega f\,dv = 0$
- 若 $\Omega$ 关于 $xOy$ 平面对称，$f(x,y,-z) = f(x,y,z)$（偶函数），则 $\iiint_\Omega f\,dv = 2\iiint_{\Omega_1} f\,dv$（$\Omega_1$ 为上半部分）
- 对 $x$、$y$ 的奇偶性同理

## 七、易错点提醒

1. **忘记写 $\rho$（柱面坐标）或 $r^2\sin\varphi$（球面坐标）**：这是最常见的扣分点！体积元素不能漏。

2. **积分上下限写反**：先确定积分顺序，内层积分的上下限是变量，外层是常数。

3. **忘记画区域图**：不做图很容易把上下限写错，尤其是交换积分次序时。

4. **截面法和先一后二混淆**：
   - "先一后二"：先积 $z$（一重），再在 $D$ 上积二重
   - "先二后一"（截面法）：先在截面 $D_z$ 上积二重，再积 $z$（一重）

5. **被积函数含 $e^{-y^2}$ 等无初等原函数**：一定要考虑交换积分次序，不要硬算。

6. **对称性用错**：奇函数在对称区域上积分 = 0，偶函数 = 2倍半区，不要搞反。
