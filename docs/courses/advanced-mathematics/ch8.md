# 第八章 向量代数与空间解析几何复习题知识点整理

说明：以下内容以《第八章 习题课》为主整理。课件中不少公式是以公式对象或图片嵌入，个别题目的原文无法百分之百完整还原；遇到这种情况，下面给出的是“按课件文本与图形信息复原后的最接近题意描述”，重点保证知识点归纳和解题思路可直接用于复习。

## 一、复习题总览

本章复习题主要覆盖四类内容：

1. 向量的坐标运算：向量的加减、模、单位向量、方向余弦、方向角、数量积、向量积、投影。
2. 平面与直线：平面方程、平面束、位置关系、夹角、距离、截距。
3. 曲面与空间区域：球面、柱面、二次曲面及其所围成立体区域的识别与作图。
4. 综合选择题：用概念判断来区分“平行/垂直/共线/共面/轨迹/曲面类型”。

## 二、按题组整理

### 题组1：已知两点坐标，求向量及相关几何量

- 题目概述：
  设点 $A(-2,3,3)$、$B(-2,2,4)$，围绕向量 $\overrightarrow{AB}$ 及三角形 $AOB$，求向量坐标、模、单位向量、方向余弦、方向角、夹角、投影以及三角形面积等。

- 对应知识点：
  - 向量坐标表示：$\overrightarrow{AB}=\mathbf B-\mathbf A$
  - 向量模长公式
  - 单位向量
  - 方向余弦与方向角
  - 数量积公式与夹角公式
  - 向量在另一向量上的投影
  - 向量积与三角形面积公式

- 为什么是这些知识点：
  这道题把“点坐标 → 向量坐标 → 长度/方向/夹角/面积”串成一条完整链条，是向量代数最标准的综合题。凡是题目出现“由两点确定向量”“求方向角”“求投影”“求面积”，几乎都要落到这几组公式上。

- 解题提示：
  1. 先把 AB 写出来，这是后续所有计算的起点。
  2. 模长、单位向量、方向余弦都建立在 AB 的坐标上。
  3. 求 $\angle AOB$ 时，不要用 $\overrightarrow{AB}$，而要用 $\overrightarrow{OA},\overrightarrow{OB}$。
  4. 求投影时记住“标量投影”和“向量投影”不要混淆，课件里主要是长度意义上的投影。
  5. 求三角形 ABO 面积时，可用 $\dfrac{|\overrightarrow{OA}	imes\overrightarrow{OB}|}{2}$，比先求夹角再代公式更稳定。

### 题组2：含参数向量的垂直与平行条件

- 题目概述：
  设 $\mathbf a=(1,3,2)$，$\mathbf b=(2,\lambda,4)$，求 $\lambda$ 的值，使得：
  1. a 与 b 垂直；
  2. a 与 b 平行。

- 对应知识点：
  - 数量积判垂直：$\mathbf a\cdot\mathbf b=0$
  - 向量对应分量成比例判平行
  - 含参数向量条件方程的建立

- 为什么是这些知识点：
  题目只问“垂直”与“平行”，本质就是在考两个最基础的判定：
  - 垂直看数量积是否为 0；
  - 平行看是否存在同一比例系数。

- 解题提示：
  1. 垂直题先列 $\mathbf a\cdot\mathbf b=0$，直接解 $\lambda$。
  2. 平行题不要只看一对分量，要让三个分量满足同一比例。
  3. 若比例关系互相矛盾，说明不存在这样的 $\lambda$；若一致，再反求 $\lambda$。

### 题组3：确定平面参数，使平面满足不同条件

- 题目概述：
  已知平面 $\pi:\ x+ky-2z-9=0$，分别求 $k$，使它满足下列条件之一：
  1. 过某已知点；
  2. 与某已知平面垂直；
  3. 与某已知平面平行；
  4. 与某已知平面成给定角；
  5. 与原点距离为给定值；
  6. 在 y 轴上的截距为给定值。

- 对应知识点：
  - 平面一般方程
  - 平面法向量 n=(A,B,C)
  - 两平面平行、垂直的判定
  - 两平面夹角公式
  - 点到平面距离公式
  - 截距的求法

- 为什么是这些知识点：
  这类题的核心不是“解方程”，而是“把几何条件翻译成法向量条件或距离条件”。平面题里，经过点、平行、垂直、夹角、距离、截距，本质上都能落到一般方程和法向量上。

- 解题提示：
  1. 一看到平面 Ax+By+Cz+D=0，就立刻想到法向量 n=(A,B,C)。
  2. 过点：把点坐标直接代入平面方程。
  3. 平行：两平面法向量平行，对应分量成比例。
  4. 垂直：两平面法向量点积为 0。
  5. 成角：用法向量夹角公式，注意有时要取锐角。
  6. 距离：原点到平面距离可直接代点到平面距离公式。
  7. 截距：令另外两个变量为 0，求对应坐标。

### 题组4：过已知直线作平面，且与另一平面成给定角

- 题目概述：
  已知一条空间直线，再求“经过该直线且与平面 $\pi$ 成 $\pi/4$ 角”的平面方程。课件解法采用了“过已知直线的平面束方程”。

- 对应知识点：
  - 空间直线的一般方程
  - 平面束方程
  - 平面法向量
  - 两平面夹角公式
  - 含参数平面的确定

- 为什么是这些知识点：
  “过一条已知直线的所有平面”不是一个平面，而是一族平面，所以必须先设平面束；之后再借助“与另一平面成定角”来确定束参数。该题本质是“平面束 + 法向量夹角”的综合应用。

- 解题提示：
  1. 若直线由两个平面相交给出，那么过该直线的平面可设为 $\Pi_1+\lambda\Pi_2=0$。
  2. 将平面束整理成一般方程，读出法向量。
  3. 再与已知平面的法向量套夹角公式。
  4. 解出 $\lambda$ 后代回即可。
  5. 若得到两个 $\lambda$，通常意味着存在两个满足条件的平面，要根据课件或题意判断是否都保留。

### 题组5：选择题（概念辨析与快速判断）

> 这一组最适合按“题型—知识点—秒杀方法”来复习。

#### 5.1 共线单位向量的数量积

- 题目概述：
  若 $\mathbf a,\mathbf b$ 为共线的单位向量，判断 $\mathbf a\cdot\mathbf b$ 的结果。

- 对应知识点：
  - 数量积公式 $\mathbf a\cdot\mathbf b=|\mathbf a||\mathbf b|\cos	heta$
  - 单位向量模长为 1
  - 共线向量夹角只能是 $0$ 或 $\pi$

- 为什么是这些知识点：
  共线 + 单位向量两个条件已经把数量积的可能值压缩到 $\cos0$ 或 $\cos\pi$。

- 解题提示：
  不要想复杂：单位向量先把模长约掉，再根据同向或反向判断数量积是 1、-1；若题目没有说明同向或反向，则应选能同时概括两种情况的表达式。

#### 5.2 向量积与原向量的位置关系

- 题目概述：
  判断 $\mathbf a	imes\mathbf b$ 与 $\mathbf a,\mathbf b$ 的位置关系。

- 对应知识点：
  - 向量积的定义
  - 向量积垂直于两个因子向量

- 为什么是这些知识点：
  这是向量积最基本的几何意义，纯概念题。

- 解题提示：
  直接记忆：$\mathbf a	imes\mathbf b$ 同时垂直于 $\mathbf a$ 和 $\mathbf b$，不与它们共线，也不与它们共面（零向量情形除外，选择题默认一般情形）。

#### 5.3 已知 $\cos\eta=0$，判断向量与哪个坐标面的位置关系

- 题目概述：
  向量与三轴正向夹角分别为 $\alpha,\beta,\gamma$，当 $\cos\eta=0$ 时判断其与坐标面的关系。

- 对应知识点：
  - 方向余弦
  - 向量与坐标轴、坐标面的垂直和平行关系

- 为什么是这些知识点：
  $\cos\eta$ 是向量与 $y$ 轴正向夹角的余弦，$\cos\eta=0$ 说明向量与 $y$ 轴垂直，进一步就能判断它平行于 xOz 面。

- 解题提示：
  记住一句话：与某轴垂直 ⇔ 平行于包含其余两轴的坐标面。

#### 5.4 已知 $\cos\gamma=1$，判断向量与哪个坐标面/坐标轴的位置关系

- 题目概述：
  当 $\cos\gamma=1$ 时判断向量 $Q$ 的方向。

- 对应知识点：
  - 方向余弦的几何意义
  - 向量与 z 轴同向时的特征

- 为什么是这些知识点：
  $\cos\gamma=1$ 表示向量与 $z$ 轴正向夹角为 $0$，即向量与 $z$ 轴同向，因此必垂直于 xOy 面。

- 解题提示：
  先判断“与哪根轴同向”，再转换成“与哪个坐标面垂直”。

#### 5.5 已知 |a±b|^2 的展开式，判断正确恒等式

- 题目概述：
  从若干选项中判断 |a±b|^2 的正确展开形式。

- 对应知识点：
  - 数量积恒等式
  - 向量模平方公式

- 为什么是这些知识点：
  这类题是在考
  $|\mathbf a\pm\mathbf b|^2=(\mathbf a\pm\mathbf b)\cdot(\mathbf a\pm\mathbf b)=|\mathbf a|^2\pm2\mathbf a\cdot\mathbf b+|\mathbf b|^2$。

- 解题提示：
  看到“模平方”就立刻改写成“自己点乘自己”，最稳。

#### 5.6 平面方程缺少某变量时的几何意义

- 题目概述：
  已知平面方程中不含 y，判断该平面的空间位置。

- 对应知识点：
  - 平面一般方程 Ax+By+Cz+D=0
  - 方程缺项的几何意义

- 为什么是这些知识点：
  平面方程不含某个变量，说明该变量对应方向可以自由变化，因此平面平行于该坐标轴。

- 解题提示：
  “缺谁就平行谁”是这类选择题的最快判断法。

#### 5.7 直线由两个平面相交给出时的位置判断

- 题目概述：
  已知直线是两个平面的交线，结合系数条件判断它是否过原点、平行于某轴、垂直于某轴等。

- 对应知识点：
  - 空间直线的一般方程
  - 两平面交线的方向向量
  - 直线与坐标轴位置关系

- 为什么是这些知识点：
  交线方向由两个平面法向量共同决定，题目考的是从方程形式快速识别直线方向特征。

- 解题提示：
  1. 可先看是否容易代入原点。
  2. 再看某个变量是否可自由取值，或利用两个法向量求交线方向向量。
  3. 选择题不必完全化为参数方程，能排除就先排除。

#### 5.8 曲面与直线的交点

- 题目概述：
  求曲面 z+xy-yz-x^2-5=0 与一条直线的交点。

- 对应知识点：
  - 直线方程参数化或代入消元
  - 曲面与直线求交

- 为什么是这些知识点：
  “交点”问题的标准做法就是把直线方程代入曲面方程，化成关于参数的一元方程，再回代求点。

- 解题提示：
  1. 先把直线写成最容易代入的形式。
  2. 代入曲面后通常得到二次或一次方程。
  3. 求出参数后别忘了回代检查每个坐标。

#### 5.9 已知球面与坐标面的截交圆，反求球面方程

- 题目概述：
  已知球面经过一点，并且与 xOy 面交成某圆，求球面方程。

- 对应知识点：
  - 球面标准方程
  - 球与坐标面的截交
  - 圆与球心、半径之间的关系

- 为什么是这些知识点：
  球与 xOy 面的交线是圆，圆心是球心在该平面上的正交投影，所以题目信息本质上是在给球心和半径施加条件。

- 解题提示：
  1. 由截交圆先确定球心前两坐标及半径关系。
  2. 再利用“球面过某点”补出剩余参数。
  3. 最后写成标准式或一般式均可。

#### 5.10 识别双叶旋转双曲面

- 题目概述：
  从若干二次曲面方程中选出表示双叶旋转双曲面的一个。

- 对应知识点：
  - 二次曲面的标准形式
  - 球面、椭球面、圆锥面、单叶/双叶双曲面的判别

- 为什么是这些知识点：
  这道题是典型“看符号判曲面类型”：双叶双曲面的标准式特征是两项为正、一项为负，且等号右端通常为 -1 或等价变形后呈双叶型。

- 解题提示：
  1. 先把方程整理成标准型或接近标准型。
  2. 看二次项符号分布，再看等号右端符号。
  3. 若系数相同或围绕某轴对称，还可顺便判断是否为旋转曲面。

### 题组6：根据边界曲面画出立体区域

- 题目概述：
  课件最后一组题要求“画出下列曲面所围成的立体区域”，共有 3 个小题。根据可识别文本，边界主要由柱面/抛物柱面、坐标面以及二次曲面或平面共同组成。

- 对应知识点：
  - 柱面、旋转曲面、二次曲面的几何形状识别
  - 曲面与坐标面的交线
  - 空间区域的边界分析
  - 立体区域草图绘制

- 为什么是这些知识点：
  这类题不要求纯代数计算，而是要求把“方程语言”翻译成“几何图形语言”。复习中最关键的是判断：
  - 每个方程表示什么曲面；
  - 每个边界从哪里截断；
  - 区域位于“上方/下方、内侧/外侧、左侧/右侧”的哪一边。

- 解题提示：
  1. 一次只看一个方程，先独立判断曲面类型。
  2. 优先分析与坐标面的交线，因为最容易画。
  3. 再找几个关键截面：例如 z=0、x=常数、y=常数。
  4. 若是“所围成区域”，要特别注意不等式方向：在曲面上方还是下方、在柱面内部还是外部。
  5. 画图时建议先画底面投影，再补高度范围。

#### 6.1 小题（1）

- 题目概述：
  由一个抛物柱面、平面 z=0，以及一个二次曲面共同围成的空间区域作图。课件文本可识别出边界中含有 x=2y^2 和 z=0，另一个上边界为二次曲面。

- 对应知识点：
  - 抛物柱面识别
  - 曲面与平面围成区域
  - 从底面投影恢复立体形状

- 为什么是这些知识点：
  x=2y^2 给出的是沿 z 方向延伸的柱面；z=0 给出底面；再配合上方二次曲面，就形成“先确定平面投影，再给高度”的典型立体。

- 解题提示：
  先在 xOy 面上画出 x=2y^2 对应的投影边界，再结合 z 的上下界补出立体高度。

#### 6.2 小题（2）

- 题目概述：
  由两个坐标面、一个平面，以及一个与 x、z 有关的曲面共同围成区域作图。课件可辨认边界中含 y=0、z=0、x+y=1，另一个边界为 x 与 z 的关系式。

- 对应知识点：
  - 坐标面作边界时的区域判定
  - 平面截取空间区域
  - 结合投影法作图

- 为什么是这些知识点：
  y=0、z=0 会把区域限制在某个象限或半空间内；x+y=1 再给出斜平面边界；最后一个曲面决定区域外形的弯曲部分。

- 解题提示：
  先在 xOy 面看 x+y=1 与坐标轴围成的三角形投影，再结合 z 方向的曲面限制画立体。

#### 6.3 小题（3）

- 题目概述：
  由二次曲面、柱面或平面 z=0、平面 x=1 等边界围成区域作图。课件可识别出边界中含 y^2=x、z=0、x=1，以及一个含 x^2、y^2、z 的曲面方程。

- 对应知识点：
  - 抛物柱面与平面联合围域
  - 顶面/底面与侧面的配合分析
  - 关键截线作图

- 为什么是这些知识点：
  这类题通常先由 x=1、z=0 等简单平面截出有限范围，再由抛物柱面或二次曲面限定侧边与顶边。

- 解题提示：
  先确定在 xOy 面上的封闭投影区域，再判断 z 是从 0 起向上延伸，还是被某个曲面截住。

## 三、复习时最该掌握的“题型—方法”对应表

| 题型 | 第一反应 | 核心公式/方法 |
| --- | --- | --- |
| 两点求向量 | 先做终点减起点 | AB=B-A |
| 求夹角 | 看是“向量与向量”还是“平面与平面” | 数量积公式 / 法向量夹角公式 |
| 求平行、垂直 | 先找法向量或方向向量 | 点积为 0、对应分量成比例 |
| 过直线作平面 | 先设平面束 | $\Pi_1+\lambda\Pi_2=0$ |
| 点到平面距离 | 套公式 | $d=\dfrac{|Ax_0+By_0+Cz_0+D|}{\sqrt{A^2+B^2+C^2}}$ |
| 曲面与直线求交 | 代入消元 | 直线参数式代入曲面 |
| 识别曲面类型 | 看方程结构和符号 | 标准型对照 |
| 画空间区域 | 先画投影，再补高度 | 交线分析 + 截面法 |

## 四、易错点提醒

1. 求 $\angle AOB$ 时要用 $\overrightarrow{OA}$ 与 $\overrightarrow{OB}$，不要误用 $\overrightarrow{AB}$。
2. 单位向量等于“原向量 ÷ 模长”，不是“每个分量都除以 2 或 3 之类的直观数”。
3. 平面平行/垂直判断对象是法向量，不是方程本身表面长得像不像。
4. 两平面夹角、直线与平面夹角都要注意公式不同，且通常取锐角。
5. “缺某变量”对应的是“平行于某轴”，这类选择题很容易因概念混淆失分。
6. 曲面作图题不要一上来画三维图，先画在坐标面上的投影最稳。
7. 识别二次曲面时，先整理到标准型附近再判断，不要被系数和移项干扰。

---

## 五、建议的复习顺序

1. 先复习向量坐标运算、数量积、向量积。
2. 再复习平面方程、法向量、平面位置关系。
3. 接着复习空间直线方程及与平面的关系。
4. 最后集中练习曲面识别和空间区域作图。

如果把本章复习题压缩成一句话，就是：

“把坐标形式翻译成几何意义，再把几何条件翻译回代数方程。”
