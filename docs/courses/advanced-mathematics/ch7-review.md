# 第七章 微分方程习题课知识点整理

## 一、复习主线

第七章习题课围绕三条主线展开：

1. 一阶微分方程的类型识别与对应解法
   - 直接积分法
   - 可分离变量
   - 齐次方程与可化为齐次方程
   - 一阶线性方程
   - 伯努利方程
   - 全微分方程与积分因子法
2. 高阶微分方程的降阶与线性方程解的结构
   - 缺项型高阶方程的降阶
   - 二阶齐次/非齐次线性方程的通解结构
3. 二阶常系数线性方程求解
   - 特征方程法
   - 待定系数法
   - 初值条件代入定常数

## 二、典型例题与对应知识点

> 说明：课件中部分公式在 PPT 转换后无法完整提取，下面的“题目概述”是根据可恢复文本整理出的最接近题意的版本，但知识点判断和解题路线是明确可靠的。

### 例1：先化简再分离变量的一阶方程

- 题目概述：原方程先经过代数变形，化成可以“分离变量”的形式，再对两边积分求通解。
- 对应知识点：可分离变量微分方程；两边积分求通解。
- 为什么是这些知识点：课件明确出现“原方程可化为……”“分离变量”“两边积分”“所求通解为”。这说明核心不在积分因子或线性结构，而在把含 x、y 的项分别放到两边。
- 解题提示：
  1. 先看方程能否整理成 $g(y)\,dy=f(x)\,dx$。
  2. 分离变量前注意定义域和可能使分母为 0 的点。
  3. 积分后别漏写积分常数；若能化简成隐式解，不必强行显式化。

### 例2：伯努利方程，代换后化为一阶线性方程

- 题目概述：题目先识别为伯努利方程，经过变量代换后化成一阶线性非齐次方程，再用常数变易法（或积分因子法）求通解。
- 对应知识点：伯努利方程；变量代换；一阶线性非齐次方程；常数变易法/积分因子法。
- 为什么是这些知识点：课件文本直接给出“伯努利方程”“一阶线性非齐方程”“利用常数变易法”。
- 解题提示：
  1. 先把方程整理成标准伯努利形式 $y'+P(x)y=Q(x)y^n$。
  2. 作代换 $z=y^{1-n}$，把非线性方程转成线性方程。
  3. 线性方程优先写出对应齐次解，再求一个特解，最后合并成通解。
  4. 回代时注意指数、绝对值以及 $y=0$ 是否也是解。

### 例3：一阶线性方程的标准化处理

- 题目概述：课件显示“将原方程写成……”，说明题目重点是把方程先化成标准形式，再按一阶线性方程处理。
- 对应知识点：一阶线性微分方程标准型；积分因子法/常数变易法。
- 为什么是这些知识点：虽然原式未完整恢复，但从前后结构看，这一题属于“先标准化，再用线性方程通法”的典型题。
- 解题提示：
  1. 先写成 $y'+P(x)y=Q(x)$。
  2. 齐次方程先解出 $y_h$。
  3. 非齐次部分可用常数变易法，也可直接使用积分因子 $e^{\int P(x)\,dx}$。

### 例4：全微分方程的两种解法

- 题目概述：已知方程是全微分方程，课件分别演示了“分项组合法”和“曲线积分法”两种求解方式。
- 对应知识点：全微分方程判别；势函数构造；曲线积分与路径无关。
- 为什么是这些知识点：课件直接写出“方程为全微分方程”“利用分项组合法求解”“利用曲线积分求解”。
- 解题提示：
  1. 先判别 $M\,dx+N\,dy=0$ 是否满足全微分条件。
  2. 若是全微分方程，就寻找势函数 $u(x,y)$，使 $du=M\,dx+N\,dy$。
  3. 分项积分时注意“只对当前变量积分”，另一变量项要通过待定函数补回。
  4. 最终答案写成 $u(x,y)=C$。

### 例5：非全微分方程，用积分因子化为全微分方程

- 题目概述：原方程不是全微分方程，需要乘一个积分因子后再积分求解。
- 对应知识点：积分因子法；可化为全微分方程。
- 为什么是这些知识点：课件明确写出“非全微分方程，利用积分因子法”。
- 解题提示：
  1. 先判断原方程不是全微分方程。
  2. 再观察积分因子是只与 $x$ 有关，还是只与 $y$ 有关。
  3. 乘上积分因子后重新检查是否成为全微分方程。
  4. 若课上常见表达式能直接凑出全微分，可优先用观察法加速。

### 例6：可降阶的高阶微分方程

- 题目概述：把某个高阶方程通过代换降阶，最后积分得到通解。
- 对应知识点：可降阶高阶微分方程；缺少未知函数 y 或缺少自变量 x 时的代换法。
- 为什么是这些知识点：课件给出“代入方程，得……故方程的通解为”，属于典型“设新变量降阶”的套路题。
- 解题提示：
  1. 若方程中未显含 y，可设 $p=y'$，把高阶方程降为关于 $p$ 的低阶方程。
  2. 若未显含 x，可设 $p=y'$ 并利用 $y''=p\,\dfrac{dp}{dy}$ 继续化简。
  3. 降阶后仍按一阶方程的分类继续处理。
  4. 最后别忘了再积分一次还原 y。

### 例7：由已知多个解反推线性方程通解，再求满足初值的特解

- 题目概述：已知某线性非齐次方程有三个解，要求求出满足给定初始条件的特解。
- 对应知识点：线性非齐次方程解的结构；对应齐次方程解的线性组合；初值条件定常数。
- 为什么是这些知识点：课件原文明确写出“由线性微分方程解的结构理论知……是对应齐次方程的解且它们线性无关，所以对应齐次方程的通解……故原方程的通解为……所求特解为……”。
- 解题提示：
  1. 非齐次方程任意两个解之差，一定是对应齐次方程的解。
  2. 从几个已知解中挑出差函数，判断是否线性无关。
  3. 写出齐次通解后，加上任取一个已知非齐次解，得到原方程通解。
  4. 再利用初始条件求常数。

### 例8：二阶常系数非齐次线性方程，右端可设待定系数特解

- 题目概述：先解特征方程得到对应齐次通解，再为右端函数设一个待定形式的特解，代回求系数，最后结合初值求特解。
- 对应知识点：二阶常系数齐次线性方程；特征方程法；待定系数法；初值问题。
- 为什么是这些知识点：课件文本依次出现“特征方程”“特征根”“对应的齐次方程的通解为”“设原方程的特解为”“由……解得”“满足初始条件的特解为”。
- 解题提示：
  1. 先求齐次部分：根的类型决定通解结构。
  2. 再根据右端 $f(x)$ 的类型设特解：多项式、指数、正弦余弦，或它们乘积。
  3. 若设出的特解与齐次解重复，要乘以 x 的适当幂。
  4. 最后再代入初值求常数。

### 例9：二阶常系数非齐次方程中的“重合”情形

- 题目概述：与例8同属待定系数法，但更像是考查“特解形式与齐次解重合时如何修正”。
- 对应知识点：特征方程法；待定系数法；特解设型的修正原则。
- 为什么是这些知识点：课件同样给出“特征方程”“特征根”“设原方程的特解为”“由……解得”，且这一类习题通常成对出现，分别练习“不重合”和“重合/共振”两种情况。
- 解题提示：
  1. 先看右端函数对应的试探特解是否已包含在齐次通解中。
  2. 若重合一次，就乘 x；若重合二次，就乘 $x^2$。
  3. 解完特解后，通解仍是“齐次通解 + 一个特解”。

### 例10：积分方程转化为微分方程的初值问题

- 题目概述：已知某函数满足含积分的关系式，题目要求求该函数；课件提示“对 x 求导两次”，把原问题转化为一个初值问题。
- 对应知识点：积分方程向微分方程转化；求导消去积分；初值条件提取。
- 为什么是这些知识点：课件明确写出“上式两边对 x 求导两次”“因此问题化为解下列初值问题”。
- 解题提示：
  1. 遇到上限为 x 的定积分，优先想到用微积分基本定理求导。
  2. 必要时连续求导，把积分项完全消掉。
  3. 原积分关系在特殊点代值，常可提取初始条件。
  4. 解出微分方程后，最好回原式验算一次。

### 例11：利用路径无关与全微分思想反求函数

- 题目概述：已知某线积分在全平面上与路径无关，并给出一条从 $(0,0)$ 到 $(1,1)$ 的积分值，要求求函数或确定其中待定表达式。
- 对应知识点：路径无关的判别与应用；势函数；全微分条件；线积分基本定理。
- 为什么是这些知识点：课件文本出现“线积分在全平面上与路径无关”“试求函数”。这类题的核心就是把积分表达式看成某个势函数的全微分。
- 解题提示：
  1. 路径无关通常意味着存在势函数 $u(x,y)$。
  2. 可先利用偏导相等条件求未知函数，再结合给出的积分值确定常数。
  3. 从 $(0,0)$ 到 $(1,1)$ 的积分值，本质上等于 $u(1,1)-u(0,0)$。

### 例12：由力学模型建立微分方程并求解

- 题目概述：根据牛顿第二定律建立运动微分方程，再解出速度、位移或其他物理量之间的关系。
- 对应知识点：实际问题建模；牛顿第二定律；一阶/二阶微分方程求解。
- 为什么是这些知识点：课件直接给出“则由牛顿第二定律得”“解此方程得”。
- 解题提示：
  1. 先受力分析，再写 $ma=\sum F$。
  2. 看未知量是时间函数，还是把速度看作位置的函数，决定方程形式。
  3. 物理背景题通常带有初始条件，如“由静止开始”。

### 例13：下沉探测仪器的速度—深度关系

- 题目概述：仪器在重力、浮力、阻力共同作用下下沉，要求建立下沉深度 y 与速度 v 的函数关系 $y=y(v)$。
- 对应知识点：微分方程建模；牛顿第二定律；链式法则把 $\dfrac{dv}{dt}$ 改写成 $v\,\dfrac{dv}{dy}$；分离变量法解初值问题。
- 为什么是这些知识点：课件原文较完整，明确要求“建立 y 与 v 所满足的微分方程，并求函数关系式 y=y(v)”；后续又写出“用分离变量法解上述初值问题”。
- 解题提示：
  1. 先列受力：重力、浮力、阻力。
  2. 因目标是求 $y$ 与 $v$ 的关系，通常把加速度写成 $\dfrac{dv}{dt}=\dfrac{dv}{dy}\dfrac{dy}{dt}=v\dfrac{dv}{dy}$。
  3. 将方程整理成关于 y、v 的可分离变量形式。
  4. 用“从海平面由静止开始”给出的条件确定常数。

## 三、习题课重点方法速查

### 1. 一阶方程先分类，再求解

拿到题后先问四个问题：

1. 能不能直接积分？
2. 能不能分离变量？
3. 能不能化成线性方程？
4. 能不能看成全微分方程，或乘积分因子后变成全微分方程？

这一章很多题，真正难点不在“算”，而在“识别类型”。

### 2. 全微分方程题要特别注意两层判断

- 第一层：原式是不是全微分方程。
- 第二层：若不是，是否存在只与 x 或只与 y 有关的积分因子。

### 3. 二阶常系数题的固定套路

1. 先解齐次方程：写特征方程，分根的情况讨论。
2. 再找非齐次特解：按右端函数类型设型。
3. 若与齐次解重合，乘 x 的幂修正。
4. 最后代入初值条件。

### 4. 解的结构类题常考“差解属于齐次方程”

若 $y_1,y_2$ 都是同一非齐次线性方程的解，则 $y_1-y_2$ 是对应齐次方程的解。这个结论在“已知几个解反推通解”的题中非常关键。

### 5. 实际应用题的核心不是公式，而是建模转换

物理应用题通常考三步：

1. 根据背景列出微分方程；
2. 通过变量替换把它化成熟悉的类型；
3. 用初始条件确定常数，得到有实际意义的表达式。

## 四、复习建议

### 容易混淆的点

- 齐次方程（形如 $y'=F(y/x)$）与线性齐次方程（形如 $y'+P(x)y=0$）不是一回事。
- “全微分方程”与“可分离变量方程”也不是同一类，判断标准完全不同。
- 待定系数法中最容易错的是“特解设型与齐次解重合时忘记乘 x”。

### 做题顺序建议

1. 先练一阶方程分类。
2. 再练全微分与积分因子。
3. 然后练二阶常系数齐次、非齐次方程。
4. 最后做解的结构题和应用建模题。

### 考前自检清单

- 是否会判断一阶方程的基本类型？
- 是否会写一阶线性方程的通解公式？
- 是否会判别全微分方程并寻找简单积分因子？
- 是否会对二阶常系数齐次方程按特征根分类写通解？
- 是否会根据右端项正确设非齐次特解？
- 是否知道非齐次方程两个解之差满足对应齐次方程？
- 是否能把简单实际问题转化成微分方程？
