## 核心考点地图

### 矩阵与行列式
- 矩阵运算（加法、乘法、转置、求逆）
- 伴随矩阵与逆矩阵的关系
- 行列式的计算（范德蒙行列式、递推行列式）
- 矩阵的秩及其性质

### 向量与线性方程组
- 向量组的线性相关性判定
- 极大线性无关组的求法
- 齐次/非齐次线性方程组的解结构
- 基础解系与通解

### 特征值与二次型
- 特征值与特征向量的计算
- 矩阵对角化的条件与正交对角化
- 二次型的标准化（正交变换法）
- 正定二次型的判定

### 证明题常见类型
- 向量组线性无关性的证明
- 矩阵可逆性的证明
- 特征向量的性质证明
