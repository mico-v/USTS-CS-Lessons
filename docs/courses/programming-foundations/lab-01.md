# Lab 01：Turtle 绘图

本实验从基础循环和 Turtle 绘图入手，练习把简单数学问题、排列组合输出和图形绘制转换为 Python 程序。

## 任务概览

1. 计算 `1 + 2! + 3! + ... + 10!` 的结果，训练循环累乘和累加。
2. 输出 5 种不同食材可能组成的健康食谱组合，练习嵌套循环和去重思路。
3. 使用 Turtle 绘制太阳花图形，理解角度、重复绘制和图形状态。
4. 附加题：计算 `1 + (1+2) + (1+2+3) + ... + (1+2+...+10)`。
5. 附加题：绘制盾牌星星类图形，进一步练习 Turtle 坐标控制。

## 训练目标

- 掌握 `for` 循环、变量累加和阶乘计算。
- 理解多层循环在组合枚举中的作用。
- 熟悉 Turtle 绘图中的移动、旋转、颜色和循环绘制。
- 将题目描述拆解为输入、处理和输出步骤。
