# Lab 05：组合数据类型

本实验训练 Python 列表、元组和字典的定义、初始化、访问、修改和删除，并用字典完成随机字符串频率统计。

## 实验目的

掌握 Python 组合数据类型的使用方法，理解列表、元组、字典各自的特点，并能根据实际问题选择合适的数据结构。

## 任务概览

1. 列表应用：创建 10 个餐厅菜单项目，练习插入、追加、扩展、排序、弹出、查找、删除、切片和遍历。
2. 元组应用：创建菜单元组，输出指定索引位置和切片范围中的元素，理解不可变序列的使用方式。
3. 字典应用：创建学生信息字典，增加年级字段，使用 `items()`、`keys()`、`values()` 遍历并删除指定键。
4. 随机字符串统计：生成 1000 个字符的字符串，统计每个字符出现次数并排序输出。

## 训练目标

- 区分列表的可变性和元组的不可变性。
- 掌握字典键值对的增删查改。
- 使用 `random`、`chr()`、`ord()` 生成测试数据。
- 用字典完成频率统计这一常见数据处理模式。
