# Lab 06：函数定义与调用

本实验通过一元二次方程求根和哥德巴赫猜想验证，练习函数定义、参数传递、返回值和算法拆分。

## 任务概览

1. 一元二次方程求根：定义 `solve(a, b, c)` 返回两个根；使用判别式 `b^2 - 4ac` 判断无实根、两个相等实根和两个不等实根三种情况。
2. 哥德巴赫猜想：定义 `isPrime(n)` 判断素数，定义 `TwoPrimes(n)` 将偶数拆分成两个素数之和，并输出 `4~100` 之间偶数的验证结果。

## IPO 要点

- 方程求根的输入是系数 `a, b, c`，处理过程是计算判别式并分支求根，输出是根或无实根提示。
- 哥德巴赫验证的输入是固定偶数范围，处理过程是遍历候选素数对，输出是形如 `24=5+19` 的分解结果。

## 训练目标

- 将可复用逻辑封装为函数。
- 理解返回多个值和 `(None, None)` 这类特殊结果的表达方式。
- 使用匿名函数或独立函数简化判别式计算。
- 通过只遍历到平方根等方式改进素数判断效率。
