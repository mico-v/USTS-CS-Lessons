# Lab 02：Python 环境与基础

本实验围绕 Python 基本输入、类型转换、数学运算和格式化输出，训练最小程序的编写与运行结果记录。

## 任务概览

1. 从键盘输入 1 个整型数和 1 个实数。
2. 将输入的实数通过四舍五入转换成整数。
3. 分别计算整型数和实数的三次方、开平方。
4. 将计算结果按保留 2 位小数的格式输出。

## 训练目标

- 熟悉 Python 解释器和脚本运行流程。
- 掌握 `input()`、`int()`、`float()`、`round()` 等基础函数。
- 练习幂运算、平方根计算和数值格式化。
- 理解不同数据类型之间转换可能带来的精度变化。
