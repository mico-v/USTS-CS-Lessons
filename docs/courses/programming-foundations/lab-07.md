# Lab 07：文件操作与 CSV

本实验训练文件读写、目录操作、源代码文本处理和 CSV 数据存取，重点是把内存中的数据持久化到文件并再次读取。

## 实验目的

理解文件、目录的操作方法，掌握 Python 文件读写方法，并能把字典类型数据保存为 CSV 格式文件。

## 任务概览

1. 源程序大小写转换：读取一个 Python 源程序文件，将除保留字、内置函数和必要导入名之外的标识符转换为大写，并生成仍可执行的新文件。
2. 字典数据写入 CSV：把学生姓名和成绩组成的字典写入 `scores.csv`，再读取 CSV 内容并打印。

## 核心知识点

- 使用 `open()` 和 `with` 管理文件读写生命周期。
- 用 `tokenize` 处理 Python 源码，避免简单字符串替换破坏语法。
- 使用 `keyword`、`builtins` 判断保留字和内置名称。
- 使用 `csv.writer` 与 `csv.reader` 完成结构化文本数据存取。
- 在 Windows 环境下注意 `newline=''` 和 `utf-8-sig` 编码对 CSV 的影响。
