# Lab 08：数据可视化

本实验从 GDP 数据出发，练习读取 CSV 数据并使用 Matplotlib 绘制直方图、折线图和饼图。

## 实验目的

理解数据展示在数据分析工作中的重要性，掌握使用 Python 第三方库进行数据可视化的方法。

## 任务概览

1. 查询并整理最近 5 年国内生产总值资料，保存为 CSV。
2. 读取 CSV，绘制国内生产总值的直方对比图。
3. 绘制 2018 年到 2022 年国内生产总值变化折线图。
4. 查询 2022 年各省市地区生产总值资料，保存为 CSV。
5. 读取地区 GDP 数据，绘制各地区占比饼图并保存图像。

## 训练目标

- 使用 CSV 文件组织表格数据。
- 使用 Matplotlib 绘制柱状图、折线图和饼图。
- 根据数据组成特点选择合适的展示方式。
- 处理中文标签、图例、标题和图片保存。
