# Lab 03：IPO 分析与流程控制

本实验要求对典型分支判断问题进行 IPO 分析，并用 Python 实现成绩处理、回文数判断和奖金提成计算。

## 任务概览

1. 成绩处理：输入 100 分制课程成绩，转换为“优、良、中、及格、不及格”五级记分。
2. 回文数判断：输入自然数，判断其各位反向排列后是否与原数相同。
3. 奖金提成：根据企业当月利润所在区间，按分段比例计算应发奖金总数。

## IPO 要点

- 成绩处理的输入是分数，处理过程是区间判断，输出是五级等级。
- 回文数判断的输入是整数，处理过程是反转或字符串对称比较，输出是真假判断结果。
- 奖金提成的输入是利润，处理过程是按利润区间累计提成，输出是奖金金额。

## 训练目标

- 使用 `if` / `elif` / `else` 描述多分支逻辑。
- 将复杂分段规则拆分为可验证的条件区间。
- 用 IPO 图或文字分析先澄清程序边界，再编写代码。
