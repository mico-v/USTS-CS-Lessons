# 技巧笔记

本页汇总课程中反复用到的 Python 基础技巧，重点是文件读写和字符串清洗。

## Python 文件读写

Python 使用内置 `open()` 处理文件读写，推荐配合 `with` 上下文管理器，确保文件在正常结束或异常发生时都能自动关闭。

常用模式包括：

- `r`：只读，文件必须存在。
- `w`：写入，文件不存在则创建，存在则清空原内容。
- `a`：追加，在文件末尾继续写入。
- `b`：二进制模式，适合图片、音频等非文本文件。
- `+`：更新模式，可读可写。

读取文件可以使用 `read()` 一次性读取全部内容，`readline()` 逐行读取，`readlines()` 返回行列表，也可以直接遍历文件对象。实际课程作业中，直接遍历文件对象通常更节省内存。

## `split()` 与 `strip()`

`split()` 用于按分隔符拆分字符串并返回列表。例如 CSV 行可以用逗号拆分为多个字段。

`strip()` 用于去除字符串两端的空白字符或指定字符，常用于清理读取文件时带入的换行符。

```python
line = "  apple, banana, orange\n"
fields = line.strip().split(",")
```

上面代码先去掉首尾空白和换行，再按逗号拆分。需要注意的是，`strip()` 只处理字符串两端，不会移除中间字段前后的空格；必要时可以对拆分后的每个字段再调用一次 `strip()`。
