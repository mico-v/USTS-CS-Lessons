# Lab 04：循环结构与游戏逻辑

本实验围绕循环结构和简单交互程序展开，要求对数列求和和猜数字游戏做 IPO 分析，并用流程图表达主要逻辑。

## 任务概览

1. 数列求和：输入元素个数 `N`，再输入逗号分隔的 `N` 个整数，分别使用 `while` 和 `for` 计算总和。
2. 猜数字游戏：程序预设或生成 `0~99` 的整数，用户循环猜测，程序根据大小关系提示“太大”或“太小”，直到猜中并输出猜测次数。

## 核心知识点

- `while` 适合条件控制的重复过程，`for` 适合遍历已知序列。
- 逗号分隔输入需要先拆分字符串，再转换为整数列表。
- 交互式游戏需要维护计数器，并在循环中根据比较结果给出反馈。
- 流程图可以帮助确认循环入口、退出条件和异常输入处理。

## 训练目标

- 熟悉两类循环结构的写法和适用场景。
- 理解输入解析、循环累加、随机数或预设值比较。
- 用流程图把重复判断过程表达清楚。
