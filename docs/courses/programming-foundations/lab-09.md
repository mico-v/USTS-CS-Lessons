# Lab 09：网页爬虫

本实验围绕网页数据采集和可视化展开，目标是抓取苏州 2022 年房价数据，并生成月份价格折线图。

## 任务概览

1. 分析安居客苏州房价页面中 2022 年各月份价格数据的结构。
2. 编写 Python 爬虫程序获取月份和房价信息。
3. 将抓取结果保存为 CSV，便于后续复用和离线分析。
4. 使用 Matplotlib 将月度房价变化绘制为折线图。
5. 当在线抓取失败时，回退读取本地 CSV 数据再绘图。

## 训练目标

- 了解网页数据抓取的基本流程：请求页面、解析内容、提取字段、保存数据。
- 使用 CSV 作为爬取结果的结构化存储格式。
- 把时间序列数据转换为折线图，观察价格随月份变化的趋势。
- 为网络失败等不稳定因素设计本地数据回退逻辑。
