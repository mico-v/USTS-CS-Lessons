# Homework 06：文本分析与数据解析

本作业包含小说人物词频统计、MNIST CSV 解析和每日英文学习词典生成。

## 任务概览

1. 读取《红楼梦》和《水浒传》文本，结合人物名单进行分词，统计主要人物出现次数并按频率排序保存。
2. 解析 MNIST 手写数字 CSV 数据，读取前 10 张 28x28 图像，并用字符方式输出像素图形。
3. 从英汉词典 CSV 中随机挑选 50 个单词，生成每日英文学习词典文本。

## 训练目标

- 使用 `jieba.add_word()` 改善中文人物名分词效果。
- 使用字典统计词频，并用 `sorted()` 按出现次数排序。
- 理解 CSV 行列结构，处理标签列和像素列。
- 综合文件读写、随机抽样和文本格式化输出。
