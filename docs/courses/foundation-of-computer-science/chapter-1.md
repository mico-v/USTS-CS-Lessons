# 第1章 信息技术概述

## 1.1 信息与信息技术

### 1.1.1 信息与数据
*   **信息 (Information)**:
    *   日常理解: 消息、知识、数据、资料。
    *   正式描述: 事物运动状态和特征的反映。
    *   表现形式: 文字、声音、图片、视频等。
    *   三要素: 物质、能量、信息。
*   **相关概念**:
    *   **信号 (Signal)**: 信息
的载体 (如电信号、光信号)。
    *   **消息 (Message)**: 信息的外壳 (如书信、语言)。
    *   **数据 (Data)**: 信息
的记录形式 (如数字、文字记录)。
    *   关系: 数据是信息的表现，信息是数据的内容。

### 1.1.2 信息处理
*   **定义**: 与信息相关的行为和活动。
*   **过程**:
    *   **收集**: 感知、测量、获取、输入。
    *   **加工**: 分类、计算、分析、转换、检索。
    *   **存储**: 书写、摄影、录音、录像。
    *   **传递**: 邮寄、出版、电报、电话、广播。
    *   **施用**: 控制、显示、指挥。
*   **人工处理的不足**: 算不快、记不住、传不远、看不清。 -> 需要信息技术 (IT)。

### 1.1.3 信息技术 (Information Technology, IT)
*   **定义**: 扩展人类信息器官功能，协助进行信息处理的技术。
*   **包含内容**:
    *   **感测与识别技术**: 扩展感觉器官 (如雷达)。
    *   **通信技术**: 扩展神经系统 (如电话、互联网)。
    *   **计算与存储技术**: 扩展大脑 (如计算机)。
    *   **控制与显示技术**: 扩展效应器官 (如显示器、机器人)。
*   **特征**: 以**数字技术**为基础，以**计算机及软件**为核心。

## 1.2 数字技术基础

### 1.2.1 比特 (Bit)
*   **定义**: Binary Digit, 二进制位。
*   **取值**: 只有 0 和 1。
*   **地位**: 组成数字信息的最小单位。
*   **存储**:
    *   电路的高/低电平 (CPU)。
    *   电容的充电/放电 (RAM)。
    *   磁化状态 (磁盘)。
    *   凹凸状态 (光盘)。
*   **计量单位**:
    *   1 Byte (字节) = 8 bits。
    *   KB ($2^{10}$), MB ($2^{20}$), GB ($2^{30}$), TB ($2^{40}$)。
    *   **注意**: 存储容量通常用 $2^{10}=1024$ 进制; 传输速率 (bps) 通常用 $10^3=1000$ 进制 (如 1Gbps = $10^9$ bps)。

### 1.2.2 比特的运算
*   **逻辑运算**:
    *   **逻辑加 (OR, $\vee$, +)**: 有1得1，全0得0。
    *   **逻辑乘 (AND, $\wedge$, $\cdot$)**: 有0得0，全1得1。
    *   **取反 (NOT, -)**: 0变1，1变0。
    *   特点: 按位独立进行，无进位。

### 1.2.3 进制转换
*   **基数 (Radix)**:
    *   十进制 (D): 基数10，逢10进1。
    *   二进制 (B): 基数2，逢2进1。
    *   八进制 (Q/O): 基数8，逢8进1。
    *   十六进制 (H): 基数16，逢16进1 (0-9, A-F)。
*   **转换**:
    *   **R进制 $\to$ 十进制**: 按权展开求和 (例如: $101.1B = 1\times2^2 + 0\times2^1 + 1\times2^0 + 1\times2^{-1}$)。
    *   **十进制 $\to$ R进制**:
        *   整数: 除R取余，逆序排列。
        *   小数: 乘R取整，顺序排列。
    *   **二、八、十六互转**:
        *   2 $\leftrightarrow$ 8: 3位合1位 / 1位拆3位。
        *   2 $\leftrightarrow$ 16: 4位合1位 / 1位拆4位。

### 1.2.4 数值表示
*   **整数表示**:
    *   **无符号整数**: 全部位用于表示数值。
    *   **带符号整数**: 最高位为符号位 (0正1负)。
        *   **原码**: 符号位+绝对值。 (0有两种表示，计算复杂)。
        *   **反码**: 正数同原码; 负数符号位不变，数值位取反。
        *   **补码**: 正数同原码; 负数反码+1。
            *   优势: 0只有一种表示; 减法可变为加法 (加负数的补码)。
            *   计算机内部主要使用**补码**。
*   **实数 (浮点数) 表示**:
    *   形式: $N = \pm S \times 2^P$ (尾数 + 阶码)。
    *   类似于科学计数法，小数点位置浮动。

## 1.3 微电子技术 (选讲)

### 1.3.1 集成电路 (IC)
*   **发展历程**: 电子管 -> 晶体管 -> 集成电路。
*   **定义**: 将晶体管、电阻、电容等元件集成在一块半导体基片(硅)上。
*   **分类 (按规模)**:
    *   SSI (小规模) < 100 元件
    *   MSI (中规模)
    *   LSI (大规模)
    *   VLSI (超大规模) > 1000万 (如现代CPU)
*   **Moore定律 (摩尔定律)**: 单块集成电路的集成度平均每18-24个月翻一番。

### 1.3.2 IC卡与RFID
*   **IC卡 (Smart Card)**:
    *   接触式 (如电话卡、银行芯片卡)。
    *   非接触式 (如公交卡、身份证)。
*   **RFID (射频识别)**:
    *   电子标签，无线通信，用于物流、门禁等。
    *   组成: 标签 (Tag) + 阅读器 (Reader)。
