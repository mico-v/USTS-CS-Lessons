# 第2章 计算机硬件

## 2.6 常用输出设备

### 2.6.1 显示器与显示卡
*   **作用**: 将计算机处理后的数字信号转换为光信号，显示文字和图形。
*   **组成**: 显示器 (Monitor) + 显示控制卡 (显卡/Video Card)。
*   **分类**:
    *   **CRT (阴极射线管)**: 也就是传统的“大头”显示器，体积大、辐射强，已淘汰。
    *   **LCD (液晶显示器)**:
        *   **原理**: 利用液晶分子的物理特性控制光线通过，需要背光源 (CCFL或LED)。
        *   **特点**: 轻薄、低功耗、无辐射、不闪烁。
*   **性能指标**:
    1.  **屏幕尺寸**: 对角线长度 (英寸)。
    2.  **屏幕比例**: 16:9, 16:10 (宽屏) 或 4:3 (普屏)。
    3.  **分辨率**: 水平像素 $\times$ 垂直像素 (如 1920$\times$1080)。
    4.  **刷新率**: 屏幕每秒更新次数 (Hz)，通常 60Hz 或更高。
    5.  **响应时间**: 像素颜色变化所需时间 (ms)，越短越好 (减少拖影)。

*   **显示卡 (显卡)**:
    *   **核心**: **GPU (图形处理器)**，专门处理图形数据，减轻 CPU 负担。
    *   **显存**: 存储待显示的图像数据。
    *   **类型**:
        *   **集成显卡**: 集成在 CPU 或主板芯片组中，共享内存，功耗低。
        *   **独立显卡**: 独立的扩展卡，有独立显存，性能强。
    *   **接口**: VGA (模拟), DVI, HDMI, DisplayPort。

### 2.6.2 打印机
*   **作用**: 将程序、数据、字符、图形打印在纸上。
*   **分类**:
    1.  **针式打印机 (Dot Matrix)**:
        *   **原理**: 打印针撞击色带。
        *   **特点**: 击打式，噪音大，精度低，速度慢。
        *   **用途**: **票据打印** (因为可以复写)。
    2.  **喷墨打印机 (Inkjet)**:
        *   **原理**: 将墨水喷射到纸上。
        *   **特点**: 非击打式，安静，能打印彩色照片，机器便宜但耗材 (墨盒) 贵。
        *   **用途**: 家庭、照片打印。
    3.  **激光打印机 (Laser)**:
        *   **原理**: 激光扫描 + 碳粉 + 感光鼓 (静电吸附)。
        *   **特点**: 速度快，精度高，噪音低，成本适中。
        *   **用途**: 办公文档打印。
*   **性能指标**:
    *   **分辨率 (DPI)**: 打印清晰度。
    *   **打印速度 (PPM)**: 每分钟打印页数。
