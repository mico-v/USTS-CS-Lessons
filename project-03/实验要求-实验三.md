# Java综合实验

## 一、实验目的

本实验旨在帮助学生掌握 Java 语言中类与对象的核心内容，通过实操训练强化代码编写与调试能力。通过本实验，学生将：

- 理解类的定义方式，掌握属性声明、构造方法重载以及封装（private 修饰符配合 getter/setter）的基本用法。
- 理解 static 关键字的含义，能够区分类变量与实例变量、静态方法与实例方法的作用范围与访问规则。
- 通过实验观察，深入理解 Java 中基本类型的值传递与引用类型的地址传递机制，明确方法调用时参数传递的实际行为。
- 掌握对象引用赋值的含义，能够区分“复制引用”与“复制对象”，理解 == 运算符对引用类型的比较规则。
- 了解静态嵌套类与内部类（非静态成员类）的定义方式、创建语法及其对外部类成员的访问权限差异。
- 通过综合性题目，将上述知识点融会贯通，培养面向对象的程序设计思维和独立调试排错的能力。

## 二、实验原理

- 继承原理：Java 通过 extends 实现继承，子类可复用父类非私有成员，支持多层继承链。
- 构造方法链：创建子类对象时先执行父类构造，再执行子类构造；super(...) 必须位于构造方法第一行。
- 方法重写：发生在父子类之间，方法签名相同，运行期动态绑定。
- 方法重载：发生在同一类中，方法名相同但参数列表不同，编译期静态绑定。
- this 与 super：this 指向当前对象；super 指向父类对象部分。
- final 关键字：final 方法可继承但不可重写。
